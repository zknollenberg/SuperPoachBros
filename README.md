# SuperPoachBros

Playable Next.js arcade prototype set in Waco.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Gameplay

- Move with **WASD** or **Arrow keys**.
- Collect golden cards for points.
- Avoid moving red obstacles.
- Survive until timer expires (45 seconds).
- Use **Restart** to start over instantly.
